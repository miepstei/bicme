Bayesian Inference for ion-channel modelling

This is the code from the recently published paper in Biophysical Journal, "Bayesian statistical inference in ion-channel models with exact missed event correction". The repo is currently private and a release branch for the paper will be made available very shortly.

In the meantime if you have interest in the code please contact michael.epstein.10 at ucl.ac.uk
